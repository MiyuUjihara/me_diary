<h2>機能と説明</h2>

<h3>ユーザーUI</h3>
<ul>
  <li>自分だけが見ることができる日記アプリ</li>
  <li>美容や健康などに関する記事を読むことができる</li>
  <li>お気に入りの記事を一覧で表示できる</li>
  <li>１日１つToDoを選択できる</li>
  <li>過去に選択したToDoの達成一覧が確認できる</li>
</ul>
